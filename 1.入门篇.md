说明：若两幅图像在相同位置上的像素点颜色的值相同，则称它们在该位置具有相同的像素点。两幅图像的相似度定义为相同像素点数占总像素点数的百分比。

```
2 2
1 0
0 1
1 1
1 1
```

之后 n 行，每行 m 个整数 0 或 1 ，表示第二幅黑白图像上各像素点的颜色。相邻两个数之间用单个空格隔开。

**输出**

一个小数，表示相似度（以百分比的形式给出，但百分号不需要显示），精确到小数点后两位。

**样例**

**输入复制**

```
3 3
1 0 1
0 0 1
1 1 0
1 1 0
```

```
44.44
```
